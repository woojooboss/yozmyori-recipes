👨‍🍳 만드는 방법

재료

오이 2개, 양파 1/4개

양념

비밀재료 3스푼, 레몬즙 1스푼, 설탕 1스푼, 소금 1스푼 (절임용), 후추 약간

만드는 법

  1. 1. 오이 절이기

    오이 2개를 얇게 썰고
    소금 1스푼 뿌려 10분 절여요.

  2. 2. 물기 짜기

    절인 오이를 꽉 짜서
    물기를 확실히 빼요. (이거 중요)

  3. 3. 양파 준비

    양파 1/4개를 얇게 채 썰어
    오이와 합쳐요.

  4. 4. 비밀재료 만들기

    비밀재료 3스푼 + 레몬즙 1스푼 +
    설탕 1스푼 + 후추를 섞어요.

  5. 5. 버무리기

    오이·양파에 비밀재료를 붓고
    버무리면 완성이에요.

한 줄 팁

반나절 냉장 숙성하면 맛이 한 번 더 좋아져요.
